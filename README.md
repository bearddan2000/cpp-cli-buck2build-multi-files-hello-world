# cpp-cli-buck2build-multi-files-hello-world

## Description
POC compile cpp programming with libraries in buck2 build tool.

## Tech stack
- buck2build
- clang
- lld

## Docker stack
- rust:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
